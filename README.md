# Yousuf — Portfolio

Personal portfolio site for Yousuf, a B.Tech AI & Data Science student at Crescent Institute of Science and Technology. Built as a single static page — no framework, no build step.

🔗 **Live:** _add your Netlify URL here after deploying_

## Highlights

- AI & Data Science coursework: algorithms, data mining, applied ML
- Featured project: **SPIS** — an AI-powered pharmacy management platform (React, Node.js, MySQL, FastAPI, Scikit-learn, Prophet)
- Algorithmic Design Lab Suite — 15 core algorithms implemented in C (sorting, searching, graph traversal, MST, shortest paths)

## Built with

- HTML / CSS / vanilla JavaScript
- Canvas-based animated graph visualization in the hero section
- Google Fonts (Space Grotesk, JetBrains Mono, Inter)

## Run locally

No build step required — just open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Deploy

Hosted on Netlify. Any static file host (Netlify, Vercel, GitHub Pages) works since there's no backend.
